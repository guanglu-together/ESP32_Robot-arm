# 🤖 ESP32 AI 视觉机械臂

**一个基于 ESP32-S3 和 PC 端 RTX 4050 (YOLOv8) 协同工作的双脑 AI 机器人项目，采用 UDP 实现极低延迟的实时控制。**

## ✨ 核心特性

- **双脑架构**：PC (RTX 4050) 负责高帧率 YOLOv8 推理，ESP32-S3 负责实时运动学控制。
- **UDP 极速通信**：抛弃传统的 TCP/HTTP 阻塞模式，采用 UDP 广播实现 <10ms 的极低延迟控制。
- **平滑插值算法**：ESP32 内部实现非线性平滑过渡，保护 MG90S 舵机免受突变信号冲击。
- **双模控制**：支持局域网 Web UI 手动控制与 AI 视觉自动追踪无缝切换。

## 🚀 未来开发路线图 (Future Roadmap)

本项目仍在持续进化中。以下是未来计划解锁的“硬核”进阶玩法：

- [ ] **✋ 手势隔空控制 (MediaPipe)**：抛弃传统的颜色/物体追踪，引入 Google MediaPipe 框架识别手部关键点。实现“捏合手指即抓取”、“张开手掌即释放”的魔法般隔空控制。
- [ ] **📐 逆运动学 (IK) 与真 3D 抓取**：从目前的“2D 视觉雷达”进化为真正的 3D 空间机械臂。引入逆运动学算法，根据目标的三维物理坐标 (X, Y, Z) 反推多个舵机的联动角度，让机械臂真正“伸出手”去触碰和抓取空间中的物体。
- [ ] **🗣️ 语音指令与大模型 (LLM) 接入**：通过 API 接入大语言模型。你可以直接对着电脑说“把桌上的手机拿给我”或“环顾一下房间”，让 AI 自动拆解任务、调用视觉模型并规划机械臂的运动轨迹。
- [ ] **📦 多目标智能分拣流水线**：利用 YOLOv8 的多类别检测能力，在桌面上放置电池、易拉罐、积木等不同物品，编写逻辑让机械臂自动识别并将其分类抓取到不同的回收区域。
- [ ] **🎥 AI Vlog 智能人脸跟随云台**：将手机夹在机械臂上，利用人脸追踪算法，让你无论走到房间的哪个角落，镜头都能自动平滑转动，永远将你锁定在画面 C 位，打造完美的桌面级智能拍摄云台。

*如果你对其中的任何方向感兴趣，欢迎提交 Issue 或 Pull Request 一起共建！*
